Detailed Description

template<typename app_traits>
class tetraphilia::color::color_detail::ByteColorCache< app_traits >

Specialization of ColorCache that implements an O(1) hash lookup for colors. The cache size is 256 entries for gray and CMYK, and 512 for RGB and Lab.
