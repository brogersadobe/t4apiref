◆ get_language_string_value()

const optional<t4::string> t4::pdf::xmp::meta_data_value::get_language_string_value ( const optional< t4::string >  language = none) const

Convenience method returns the value as a string in a particular language.

Returns 'none' if the requested property doesn't exist, or if it does exist but not in the requested language. This includes the unlikely case of an empty array. By definition the default language always exists (in a non-empty array) so that a request for 'none' or for "x-default" always returns a value.

Throws if the property is not a valid "Language Alternative," that is, an alternative array of simple text items each with an an xml:lang qualifier. See section 8.2.2.4 "Language Alternative."

Parameters
languageThe language to look for. If omitted, then the first (default) item in the alternative array is chosen.
Returns
The value of the property for the specified language, or 'none' if the property or language doesn't exist.
