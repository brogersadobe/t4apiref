Detailed Description

Class representation of a name tree. See PDF Reference section 3.8.5 "Name Trees."

Values can be accessed either by name or by index.

This class is not exposed as part of the SDK and instead is an internal class. SDK classes that use name trees will wrap a name_tree instance. See for example the resources class.
