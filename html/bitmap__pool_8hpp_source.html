38 /*
39  Class that manages a resource pool of client-provided bitmaps.
40 
41  BitmapImplementation must be noexcept-movable, and noexcept-default-constructible.
42 
43  You must also insure that the following expression compiles:
44 
45  size_t x = get_bitmap_size( bitmap );
46 
47  (where bitmap is a BitmapImplementation& ) and returns the bitmap's approximate size in
48  bytes. Use ADL if you can to make sure this expression compiles.
49 
50  Use bitmap_factory::create() to create a std::shared_ptr<BitmapImplementation> which
51  corresponds to a cache entry. Your BitmapImplementation instances will be purged
52  in accordance with the thresholds specified in bitmap_factory's ctor.
53 
54 */
