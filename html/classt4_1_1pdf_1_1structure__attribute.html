◆ get_text_string_value()

t4::string t4::pdf::structure_attribute::get_text_string_value ( ) const

Convenience method for text string attributes, returns the text string's value as a t4::string. Throws corrupt_data if the value is not a PDF string object. Behavior is undefined if the string is a byte string. (Might work if the string is an ASCII string.) For a discussion of PDF string types, see PDF Reference 2.0 Section 7.9.2 "String object types."
