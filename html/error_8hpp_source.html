67 /*
68  * Wrapper for a block which catches exceptions and translates them to NSError.
69  * Example usage:
70  * auto foo = t4::call_with_error( &error, [&]()
71  * {
72  * return t4::function_that_may_throw_exception( arg1, arg2 );
73  * } );
74  * The [&] indicates that captured variables (e.g arg1, arg2) are treated as references.
75  * @param error Reference to NSError* if you wish to have exception translated to NSError, nil otherwise.
76  * If Non-null, will be populated with NSError (T4ExceptionConversion) category which may have domain
77  * value T4ErrorDomain, in which case it uses T4ErrorCode as its code value, and has a exceptionPtr
78  * property with value being the originating t4::exception_ptr.
79  * @param f Callable type to execute. Should be "simple" such that return type can be inferred by
80  * std::result_of. You'll get a compiler error otherwise.
81  */
