Detailed Description

Class representation of an embedded file stream for the special case where it holds the embedded file for an associated file. Unlike the general case for embedded files, the Subtype key is required.

t4::string t4::pdf::associated_embedded_file::get_subtype ( ) const

Returns the value of the required "Subtype" key, which is the MIME type of the file's data.

Does not perform any validation on the value. However for certain valid but deprecated MIME types this will return instead the supported, canonical equivalent.

Returns
The value of the "Subtype" key.
