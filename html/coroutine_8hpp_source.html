43 /*
44  Wrappers around boost::coroutine which do two things on every context switch:
45 
46  1) Switch the TLS's notion of the "stack top" so that calls to t4::check_stack_depth()
47  work with reasonable values, and
48  2) Switch the TLS's notion of the current thread heap, so that T4 coroutines can use
49  t4::get_thread_heap() and make reasonable assumptions about the timing of the destruction
50  of objects in that heap.
51 
52  These types work exactly like the boost::coroutines2::push_type/pull_type.
53 */
