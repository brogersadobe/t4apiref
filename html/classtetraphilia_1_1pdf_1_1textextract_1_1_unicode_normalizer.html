Detailed Description

Provides unicode normalization services for search.

A unicode normalizer's job is to convert unicode characters into "canonical" characters (perhaps more than one). The types of things which it can do are: 

- ligatures -> sequence of simple characters
- alternate glypj sets -> standard glyphs sets (half width -> full width katakana)
- potentially full unicode normalization according to spec

Also it may be asked to do: 

- conversion to canonical case (usually lower, but not required)
- punctuation normalization (open quote / close quote -> symmetric quote)

Haiku provides the "DefaultUnicodeNormalizer" class which should be sufficient for most uses.

The normalizer is used for searching. It needs one templatized member method:


    class UnicodeNormalizer
    {
        template<typename unicode_sequence_storage_type>
        void AppendNormalizedChar(unicode_sequence_storage_type& seq, std::uint32_t c, bool normalizeCase, bool normalizePunctuation, bool normalizeHalfwidthKatakana) const;
    };
