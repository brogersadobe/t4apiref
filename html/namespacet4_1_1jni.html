◆ get_enum_value()

jobject t4::jni::get_enum_value ( JNIEnv *  env,
const char *  enum_class_name,
const char *  value_name 
)

Returns a value from a Java enum. Throws on failure.

This is less efficient than doing the equivalent inline. For one thing, there is string manipulation to create what would likely be a string literal if done inline. Also there is a lookup of the jclass, rather than reusing a single instance retrieved once. But the convenience of having this common idiom implemented once can be worth these small inefficiencies.

Parameters
enum_class_nameFully qualified enum class name (using "/" rather than ".").
value_nameName of the enum value.
