Detailed Description

Class representation of an embedded file stream.

This is a base class and is typically not used directly by clients. Instead use derived classes embedded_file and associated_embedded_file.

void t4::pdf::embedded_file_base::write_bytes ( vector< unsigned char > const &  buffer)

Writes binary data into the PDF stream. Overwrites any existing data.

Sets the following properties:

  • Filter (hard coded to "FlateDecode")
  • Length
  • Params/Size
  • Params/CheckSum
  • Params/ModDate

Preserves any existing Params dictionary (including its CreationDate value).
