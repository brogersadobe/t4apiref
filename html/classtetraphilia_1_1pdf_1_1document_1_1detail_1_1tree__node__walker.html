Detailed Description

template<typename tree_traits, typename kids_func, typename names_func>
class tetraphilia::pdf::document::detail::tree_node_walker< tree_traits, kids_func, names_func >

Walks the nodes of a name or number tree and invokes a caller supplied callback for each node encountered. Leaf nodes are included in this enumeration, but the individual items in leaf nodes are not stepped through.

Template Parameters
tree_traitsTraits encapsulating name tree vs. number tree functionality.
kids_funcCallback invoked on kids arrays.
names_funcCallback invoked on names/nums arrays.
