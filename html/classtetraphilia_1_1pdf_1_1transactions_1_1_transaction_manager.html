◆ rollbackLastTransaction()

void tetraphilia::pdf::transactions::TransactionManager::rollbackLastTransaction ( )

Completely rolls back the last transaction (initiated using the StartTransaction call). All changes made as a part of the last transaction are reverted and the transaction is no longer a part of the undo stack. Note that any undo / redo functors passed to StartTransaction are not called.
