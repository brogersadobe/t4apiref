Detailed Description

Class representing a structure value. See section 6.3.3 "Structure values."

This class doesn't really add value, since it just wraps a value_map. Instead of having this class we could just use a value_map directly. But a class is provided here for symmetry with the other basic XMP types: simple_value and array_value.
