Detailed Description

template<typename app_traits>
class tetraphilia::pdf::text::UTF32StringConstIterator< app_traits >

Iterator class that walks a String's sequence of UTF-32 character "code point" values.

This class is appropriate for strings noted as "text" in the PDF Reference Manual (see PDFRM 1.5, 3.8.1 "Text Strings"). In PDF, "Text" strings are encoded either in PDFDocEncoding or Unicode using UTF-16BE (big endian). Unicode strings are identified by the two leading characters 0xFE, 0xFF; all other strings are PDFDocEncoding. This iterator converts such strings and returns their UTF-32 unicode code point representation.

The UTF32StringConstIterator class produces UTF-32 characters independent of the string's encoding. It skips over language/country escape sequences as well.
