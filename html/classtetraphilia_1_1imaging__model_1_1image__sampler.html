Default constructor. This does not construct a meaningful image_sampler. Maps the portion of the srcImage that is specified by srcImageBounds into portion of the dstImage that is specified by dstImageBounds.

Parameters:

iq describes quality levels for bitmap image sampling - kIQMedium* family will employ cubic sampling.

sf describes sharpening levels for bitmap image sampling

transform describes the total device matrix

dstImageBounds describes optional output image bound - where within the dst you want it to write the image into.

srcImageBounds input source image bound - where within the src you want it to read the image from.

dstImage describes optional output destination image - the bits that it writes the resampled image into.

srcImage describes input source image - the bits that it reads the source image from.

constraints description of alignment constraints

origMeasureOfMatScaleFactor description of "starting measure of matrix scale factor" Speedo performs auto smoothing and auto sharpening when scaling up or down images. Speedo uses the "measure of scaling factor" to fully automate this process. There are times that an image might go through multiple transformations (i.e. we might break the "transform" into two matrices) and use a different "quality" setting. In these cases, to produce the best possible looking image the system needs to know the origMeasureOfMatScaleFactor that was computed from the "origTransform".

Return Value:

None.
