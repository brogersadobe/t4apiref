optional<t4::pair<t4::string, structure_namespace> > t4::pdf::structure_role_map::get_mapped_value ( const t4::string &  base_value,
bool  do_multilevel_mapping 
) const

Returns a mapped value for a specified base value. If do_multilevel_mapping is false it returns the first level mapping and the clients are expected to drive multi-level mapping. For e.g. if <A, NS1> -> <B, NS1>, and <B, NS1> -> <C, default-namespace>. In this case get_mapped_value("A") will return <B, NS1>. The client is the expected to get RoleMapNS and create a new instance of rolemap and the use it to get the mapping for B to finally get <C, default-namespace>. If do_multilevel_mapping is true, it will do a multi-level mapping. That is, in the previous example, get_mapped_value("A") will return <C, default-namespace>.

Parameters
base_valueSpecifies the base value for which to return a mapped value.
do_multilevel_mappingSpecifies if the mapping should be multi-level mapping or just first level mapping.
Returns
A pair whose first element is the mapped value for the specified base value and the second value is the namespace
