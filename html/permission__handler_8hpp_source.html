32 /* This class handles all the document level permissions other than those enforced and handled by security handlers. We currently don't have multiple permission handlers like
33  * Acrobat does because that would be an overkill considering our current needs. If the need arises in future, we can extend this class to implement multiple permission handlers and
34  * the relevant code (in this class) can be easily split across them - it's modular enough to do that without any extra work. */
