41 /*
42  Select between boost::thread and std::thread and populate the t4 namespace accordingly.
43 
44  boost::thread has thread interruption support, while std::thread (per the C++11 spec) does not.
45  This makes boost::thread more desirable than std::thread. However, there are platforms
46  (well, one, WinRT) that support std::thread but to which boost::thread has not yet been
47  ported. Thread interruption is not available on these platforms, but the semantics of
48  thread usage are still basically the same.
