typedef std::pair<std::uint32_t,text::CharCodeLookup> tetraphilia::pdf::text::code_point

Holds the key and value of one entry in a CharCodeMap, as returned by CharCodeMap::GetCodePoints().

The first value is the character code for the entry (that is, a value that appears in a content stream). The second value is the CharCodeLookup for that entry.
