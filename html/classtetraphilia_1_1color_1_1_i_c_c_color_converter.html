Detailed Description

template<typename signal_traits>
class tetraphilia::color::ICCColorConverter< signal_traits >

This is the publicly-accessible color converter for converting colors between ICC color spaces. It is templatized on the signal traits type, contains a cache of converted colors, and contains a private instance of ICCColorConverterImpl (the non-public converter type that deals only with the internal color conversion type). The ICCColorConverterImpl instance is cached on the source color space.
