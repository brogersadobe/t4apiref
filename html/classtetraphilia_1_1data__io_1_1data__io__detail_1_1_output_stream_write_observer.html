Detailed Description

An instance of this class serves as an observer to the incoming "write" operations to another OutputStream instance, allowing a client to monitor those writes through invocation of arbirary client-provided callback.

This facility can be used for logging or computing a rolling hash over the data being written to an OutputStream.
