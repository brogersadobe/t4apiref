27 // StackBasedObjectPointerPusher and StackBasedObjectValuePusher
28 //
29 // Utility template for exception safe mechanism using the execution stack to push and pop objects into an owning object
30 // To use:
31 // 1) create a class pushpop_helper with two methods GetCurrent and SetCurrent
32 // GetCurrent returns the current value of the object in the owning object
33 // SetCurrent sets the current value of the object in the owning object
34 // 2) Place a templatized StackBasedObject*Pusher on the stack with the 3 template parameters and 2 runtime parameters
35 // template parameters - the helper class, the owning object class, the pushed object class
36 // runtime paramaters - the owning object and the value to be pushed
37 // 3) At destruct time this object will restore the original value into the owning object
38 //
39 // TODO: Can these two templates should be merged into one? Couldn't quite figure it out
40 //-----------------------------------------------------------------------------
