Maps the portion of the srcImage that is specified by srcImageBounds into portion of the dstImage that is specified by dstImageBounds.

Parameters:

dstImage describes optional output destination image - the bits that it writes the resampled image into.

srcImage describes input source image - the bits that it reads the source image from.

dstImageBounds describes optional output image bound - where within the dst you want it to write the image into.

srcImageBounds input source image bound - where within the src you want it to read the image from.

iq describes quality levels for bitmap image sampling - kIQMedium* family will employ cubic sampling.

sf describes sharpening levels for bitmap image sampling

Return Value:

true if successful; false otherwise.
