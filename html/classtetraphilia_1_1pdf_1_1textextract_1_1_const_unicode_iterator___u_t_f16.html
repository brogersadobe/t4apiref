Detailed Description

template<typename const_utf16_iterator_type>
class tetraphilia::pdf::textextract::ConstUnicodeIterator_UTF16< const_utf16_iterator_type >

Class concept that provides forward iteration services over a source of unicode code points, independent of encoding and storage mechanisms.

A model of the _CONCEPT_unicode_sequence concept must provide the following:

const_forward_iterator

A data type that provides basic forward iteration services over a sequence of 32-bit unicode values.

For example:


struct unicode_sequence
{
    class const_forward_iterator
    {
    public:
        const_forward_iter( const const_forward_iter& );
        voic operator=( const const_forward_iter& );
        bool operator==( const const_forward_iter& ) const;
        bool operator!=( const const_forward_iter& ) const;

        template<typename int_type>
        void operator+=( int_type i );  // i >= 0

        template<typename int_type>
        std::uint32_t operator[]( int_type i ) const;   // i >= 0

        std::uint32_t operator*() const;
    };

begin, end

Methods that provide instances of const_forward_iter that may be used to enumerate all the unicode characters in the sequence.


    const_forward_iterator begin() const;
    const_forward_iterator end() const;
};
