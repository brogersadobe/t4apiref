84 // When destructed, a FixedSizedAllocator (defined below) deletes the blocks
85 // that it allocated. Blocks allocated by AFFixedSizeAllocator (defined above)
86 // are never explicitly deallocated. There is no AFFixedSizeAllocator object.
87 // AFFixedSizeAllocator::allocate and AFFixedSizeAllocator::deallocate
88 // remove and return to a global free list. In constrast,
89 // FixedSizeAllocator::allocate and FixedSizeAllocator::deallocate
90 // remove and return to a free list that belongs to the FixedSizeAllocator
91 // object, and a FixedSizeAllocator objects keeps track of the blocks
92 // it allocated so that the blocks can be deallocated when the
93 // FixedSizeAllocator is destroyed. The code below was written by Scott Cohen,
