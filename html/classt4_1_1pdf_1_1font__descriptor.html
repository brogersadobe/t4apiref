◆ get_font_weight()

optional<t4::real> t4::pdf::font_descriptor::get_font_weight ( ) const

Return the value of the optional FontWeight key.

Note that although the property is specified as a number in the PDF spec, the only legal values are even multiples of 100 between 100 and 900. This method returns the actual number and performs no validation or adjustment of values.

◆ get_program_type()

optional<program_type> t4::pdf::font_descriptor::get_program_type ( ) const

Returns the format of the embedded font, or none if there is no embedded font.

The embedded font program is found in the FontFile, FontFile2 or FontFile3 stream in the FontDescriptor dictionary.

The value will be other than none if and only if has_embedded_font() returns true.

Returns
The format of the font program present in the font dict.
