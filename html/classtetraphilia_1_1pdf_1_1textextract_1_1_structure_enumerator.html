Detailed Description

template<typename app_traits>
class tetraphilia::pdf::textextract::StructureEnumerator< app_traits >

Object that enumerates content ranges in a structure object.

The enumerator, whose work is performed on a so-called driver thread, maintains a worker thread which performs the actual enumeration.
The driver thread spawns a new worker thread either when the first content range is enumerated, or when a content range that we have already passed in the display list needs to be enumerated. In the latter case, the worker thread in existence at the time is terminated before a new worker thread is created. Every time a content range needs to be enumerated, the driver thread signals m_newRangeEvent, then waits on the m_rangeDoneEvent. In turn, after enumerating a content range, the worker thread calls the HandleRangeDone callback in the enumerator, which signals m_rangeDoneEvent and then waits on m_newRangeEvent.
