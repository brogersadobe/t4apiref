Detailed Description

template<typename app_traits>
class tetraphilia::pdf::text::UTF16StringConstIterator< app_traits >

Iterator class that walks a String's sequence of UTF-16 word units (NOT characters!).

In most cases each word unit represents a character, but UTF-16 also has 
a construct known as "surrogate pairs" where a pair of UTF16 word units 
comprise a single character.

This class is appropriate for strings noted as "text" in the PDF
Reference Manual (see PDFRM 1.5, 3.8.1 "Text Strings").  In PDF, "Text"
strings are encoded either in PDFDocEncoding or Unicode using UTF-16BE (big endian).
Unicode strings are identified by the two leading characters 0xFE, 0xFF;
all other strings are PDFDocEncoding.

The UTF16StringConstIterator class produces UTF-16 characters independent
of the string's encoding. It skips over language/country escape
sequences as well.

<b>Intended Usage</b>

The following code fills a buffer of unsigned shorts with UTF16 
data representing the contents of an arbitrary text string from a PDF file:


   // Fill the given buffer with the UTF-16 values representing the given
   // string, whether the string is encoded in Unicode (UTF-16BE) or PDFDocEncoding.
   // A nul terminator is always placed at the end of the string or at least 
   // after as much of the string fits in the buffer.
   // Stops when the string terminates or no additional *characters* can be 
   // fit into the string (thus, a surrogate pair will NOT be partially copied).
   // Returns the total number of UTF-16 values (not characters) generated (not including the terminator). 
   // Thus, the return value will be at most destLen-1.
   // IMPORTANT: if the return value is destLen-2, the returned the string could still be truncated!
   template<typename object_traits>
   size_t FillBuffer( const String<object_traits>& str, unsigned short* dest, size_t destLen )
   {
      const_unicode_iterator<object_traits> it( str );
      const unsigned short* p = dest;
      const unsigned short* dstEnd = dest + destLen;

      while ( !it.Done() && p < dstEnd )
      {
         *p++ = *it;
         ++it;
      }

      if ( !it.Done() && p > dest && 0xDC00 <= *it && *it <= 0xDFFF )
         --p; // don't include previous value as (if the data is not corrupt) it was the first of a surrogate pair (assuming the data is valid UTF-16)

      *p = 0;
      return p - dest;
   }
