185  /*
186  Get the aligned pointer result, i.e, the value returned by the call to t4::malloc().
187 
188  This is guaranteed to be aligned to the originally requested block. Note that since
189  "this" is necessarily aligned to alignof(alloc_preamble), so is this+1, which
190  satisfies all requested alignments less than alignof(alloc_preamble). If the
191  requested alignment is larger, we know that we got the base address of the
192  alloc_preamble by walking backwards from an address aligned to that value.
