Create a color converter.

This method creates a color converter that converts from this color space to the specified destination color space.

Note that gcc 2.95 is not able to deduce the template argument, so you will need to use the following obscure syntax if you need to support that platform:


const ColorSpace & srcCS = ...
typename ColorConverter<signal_traits>::ptr_type converter =
    srcCS.template CreateConverter( dstCS, renderIntent );
Parameters
linearIf is set to true, the converter will convert colors into a linearized color space. This permits linearized blending, after which you may call one of the ColorConverter::Unlinearize methods to complete the color conversion process by applying the gamma correction.
