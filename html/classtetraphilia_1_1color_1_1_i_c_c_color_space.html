◆ HasCachedOutputStage()

template<typename app_traits >
bool tetraphilia::color::ICCColorSpace< app_traits >::HasCachedOutputStage ( RenderIntent  renderIntent) const
inline

These next three methods support caching the output stage of a color transformation engine. For a given destination profile and rendering intent, the output stage will be the same for any transformation that uses that destination space. This allows us to avoid regenerating those stages in some cases. This is especially helpful when dealing with the monitor profile.
