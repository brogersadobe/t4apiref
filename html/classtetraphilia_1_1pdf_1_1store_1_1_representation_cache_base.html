Detailed Description

template<typename app_traits, typename representation_type>
class tetraphilia::pdf::store::RepresentationCacheBase< app_traits, representation_type >

THIS CLASS IS NOW DEPRECATED. Please use: 

 t4::pdf::representation_cache

instead.

Cache of "representations" of PDF dictionaries from a single Store.

Dictionaries in PDF represent many things: pages, fonts, color spaces, annotations, images, etc. The RepresentationCache mechanism provides a way for users of an Store to provide "representations" of PDF dictionaries as various types of objects.

Intended Use

Representation classes are intended for use in contexts where a client has a PDF dictionary and wants a piece of expensive-to-compute data derivable from the dictionary. Examples include:

  1. Given a dictionary that is known to be a font dictionary, get a font that can be used to draw glyphs from that font.
  2. Given a color space dictionary, get a ColorSpace object that implements that color space.

Representation instances are accessed via AccessRepresentation.
