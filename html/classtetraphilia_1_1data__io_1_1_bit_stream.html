Detailed Description

A simple class having services to read a stream of bits from a buffered_stream and break them up into numbers of varying bit sizes. Upon construction the bit_stream object is conceptually pointing to the high-order bit of the byte pointed to by the buffered_stream given to it. Routines like GetNextNBitsAsInt read the next 'n' bits from the stream as an integer. Upon completion the routine advances its current point into the stream by 'n' bits. Note that the data pointed to by the stream will be considered to contain "big-endian" data (for any case where GetNextNBitsAsInt is asked to return an integer with more than 8 bits, or any integer that crosses a byte boundary in the stream data).

bit_stream's also have a few optimized versions of these routines for cases where you know that the integer you are looking for in the stream starts on a byte boundary.
