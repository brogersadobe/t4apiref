Detailed Description

template<typename app_traits>
struct tetraphilia::pdf::textextract::TextSelectionRectangle< app_traits >

Data structure that helps compute a text selection based on two mouse locations in a possibly multi-page view.

This structure takes two page IDs and two subrectangles (in default user space) on those pages. If the two page IDs are the same, the selection takes place entirely on one page within the first rectangle, and the second rectangle is ignored.

If the selections ends on a different page from the start, two different rectangles are required, which identify for each page a mouse-up or mouse-down point, and (typically) one of the page's four corners. This corner point identifies a directional bias in which to look for content points in the very common case when A does not in fact intersect a glyph bounding box.

An example: 

                +----------+
                |          |
                |          |
                |          |
                |     A    |
                |          |
                +-----1----B

   +----------+ +----------+
   |          | |          |
   |          | |          |
   |          | |          |
   |          | |          |
   |          | |          |
   +-----2----+ +-----3----+

   D----------+  A = mouse down point
   |          |  B = bottom right of mouse down rect
   |          |  C = mouse up point
   |     C    |  D = top left of mouse up rect
   |          |
   |          |
   +-----4----+

The correct result for this is a content range starting at the first glyph in read order that is between A and B, including all the text on page 2 and 3, and terminating at the last glyph in read order between C and D.

In the common case where the mouse down and mouse up are the same point, this will identify an empty content range adjacent to the glyph nearest that point.

Note that page rotation in the user interface will affect the computation of the rectangles. The correct corner rectangles are not always the bottom right of the first page and the top left of the last page.
